A [TypeScript](https://www.typescriptlang.org/) library that wraps the [postMessage API](https://developer.mozilla.org/en-US/docs/Web/API/Window/postMessage)
to provide structured, type-safe messaging between iframes, windows, and browser tabs.
A shared secret is used to scope messages so that unrelated postMessage events are ignored.

The API documentation is available at [friedemannsommer.github.io/typescript-postmessagehandler][docs].

The source code for this project can be found on GitHub at [friedemannsommer/typescript-postmessagehandler][repo].
