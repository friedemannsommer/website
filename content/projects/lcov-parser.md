A [LCOV](https://github.com/linux-test-project/lcov) trace file parser written in [TypeScript](https://www.typescriptlang.org/).
It supports synchronous and asynchronous parsing and accepts `string`, `Buffer`, `ArrayBuffer`, and `Readable` streams as input.

The package is available on npm as [`@friedemannsommer/lcov-parser`](https://www.npmjs.com/package/@friedemannsommer/lcov-parser).

The full API documentation can be found at [friedemannsommer.github.io/lcov-parser][docs].

The source code for this project can be found on GitHub at [friedemannsommer/lcov-parser][repo].
